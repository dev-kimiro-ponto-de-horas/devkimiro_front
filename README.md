## 🚀Rodar o Projeto

- Para fazer o projeto rodar:

```bash
  npm install
  npm i ng-spring-boot
  ngs --open
```

## 🛠️ Ferramentas Utilizadas

### Back-End:

- Spring Boot
- Spring Framework  
- MySQL
- Postman

### Front-End:

- Crescer e se implementarmos em empresas com o decorrer dos anos
